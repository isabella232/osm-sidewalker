Fixing Sidewalks

This tool is designed to assist in fixing sidewalk tagging in OpenStreetMap. For more on this problem, see "Updating insufficiently mapped sidewalks in OpenStreetMap"

This project is powered by Tile Reduce, Turf and OSM QA Tiles. To contribute to mapping sidewalks, follow along here. To contribute to the sidewalk analysis Tile Reduce code, visit the github page

Click a highlighted footway segment for information about that segment. To open an entire tile's worth of ways in JOSM, click inside the desired tile but not on a highlighted footway.
